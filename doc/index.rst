verify_encat
============

:Release: |release|
:Date: |today|

SAML IdP test utility for entity categories. The service can verify how a IdP responds dependent on which entity
categories that are defined.

You can test IdP's that are member of the SWAMID federation at: https://samltest.swamid.se
